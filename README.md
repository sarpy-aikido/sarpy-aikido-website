# Sarpy Aikido Website

Static website for Sarpy Aikido, a martial arts dojo in La Vista, Nebraska offering practical aikido training since 1998.

## Pages

- **index.html** - Homepage with classes overview and location
- **about.html** - Mission, values, and instructor info
- **classes.html** - Full class schedule and training details
- **lineage.html** - Aikido lineage from O-Sensei to present

## Development

No build tools required. Open any HTML file in a browser.
